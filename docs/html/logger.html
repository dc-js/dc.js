annotate( [fn] [, msg])

Used to provide an informational message for a function. It will return a wrapped version of the function, which will will issue a messsage with stack when invoked. The message will be issued only once.

Parameters:
Name Type Argument Description

msg String <optional>

Example
_chart.interpolate = logger.annotate(function (interpolate) {
   if (!arguments.length) {
       return _interpolate;
   }
   _interpolate = interpolate;
   return _chart;
}, 'lineChart.interpolate has been annotated since version 3.0 use lineChart.curve instead');
